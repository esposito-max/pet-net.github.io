Sobre o PetNet

O PetNet é uma plataforma integrada desenvolvida para conectar ONGs, protetores de animais, prestadores de serviços e adotantes. O objetivo é facilitar a adoção responsável e centralizar serviços essenciais para pets.

Tecnologias utilizadas: Figma (Prototipagem), Flutter, Django, GitHub Pages.
